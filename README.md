## 📝 Descripción del Proyecto

GameVault es una aplicación móvil de catálogo para una tienda de videojuegos desarrollada para el taller práctico de Navegación y Hooks. La aplicación permite explorar títulos, consultar detalles técnicos, registrar nuevos videojuegos mediante formularios controlados y visualizar noticias con actualización automática.

## 📦 Instalación y Ejecución

1. **Clonar el repositorio:**
   ```bash
   git clone [https://github.com/TU_USUARIO/GameVault.git](https://github.com/TU_USUARIO/GameVault.git)
   cd GameVault
   ```
2. **Instalar dependencias:**
   ```bash
   npm install
   ```
3. **Iniciar Metro Bundler:**
   ```bash
   npx react-native start
   ```
4. **Ejecutar en Android:**

   ```bash
   # Para Android
   npx react-native run-android

   # Para iOS
   npx react-native run-ios
   ```

5. **Limpiar el caché (en caso de errores):**
   ```bash
   npx react-native start --reset-cache
   ```

## 🛠️ Tecnologías Utilizadas

- **React Native CLI**
- **React Navigation** (Stack & Bottom Tabs)
- **JavaScript (ES6+)**
